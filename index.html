Hello World, Notes to write in a commit:
  • feat - a new feature
  • fix - a bug fix
  • docs - changes in documentation
  • style - Any change done in styling
  • refactor - Code changes that neither fixes a bug or adds a feature
  • test - everything related to testing
  • chore - updating build tasks, package manager configs, etc
What is GIt?
It is a commit tracking mechanism or change tracking mechanism.
What is GITHUB?
You put that code base on the web.
